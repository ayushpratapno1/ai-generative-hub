# AI Generative Hub

A comprehensive Django-based web application that combines AI-powered image generation and intelligent chatbot functionality. This platform leverages multiple AI models to provide users with creative image generation capabilities and conversational AI assistance through a user-friendly web interface.

## 🌟 Features

### Image Generation
- **Prompt-based Image Generation**: Generate high-quality images from text descriptions
- **Dual AI Models**: 
  - **Blackbox Model**: Advanced image generation with creative flexibility
  - **Fusion Model**: Enhanced image synthesis with improved quality
- **Real-time Generation**: Instant image creation through API integration
- **User-friendly Interface**: Intuitive web-based image generation tools

### Chatbot Functionality
- **Intelligent Conversations**: AI-powered chatbot for natural language interactions
- **Context-aware Responses**: Maintains conversation context for better user experience
- **Multi-purpose Assistance**: Helps with various queries and tasks
- **Real-time Communication**: Instant responses through optimized API calls

### Platform Features
- **Web-based Access**: No installation required - access through web browser
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Secure API Integration**: Robust backend with secure API handling
- **Production Ready**: Deployed and optimized for live usage

## 🛠️ Technology Stack

- **Framework**: Django (Python web framework)
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Django REST Framework for API handling
- **AI Models**: 
  - Blackbox AI for image generation
  - Fusion AI for enhanced image synthesis
  - Custom chatbot integration
- **Deployment**: Render cloud platform
- **Database**: Django ORM with optimized queries

## 💡 How It Works

### Image Generation Process
1. **User Input**: Enter descriptive text prompts for desired images
2. **Model Selection**: Choose between Blackbox or Fusion AI models
3. **API Processing**: Backend processes the request through selected AI model
4. **Image Delivery**: Generated image is displayed instantly in the web interface

### Chatbot Interaction
1. **Message Input**: Type questions or start conversations
2. **Context Analysis**: AI processes the message with conversation context
3. **Response Generation**: Intelligent response generated through API
4. **Real-time Display**: Immediate response display in chat interface
